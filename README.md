# Menjalankan Project

## Clone Project
```bash
git clone https://github.com/ilhaamms/be-live-ilham-muhammad-sidiq.git
```

## Copy sample.env to .env
```bash
cp sample.env .env
```

## Start Program
```bash
go run main.go
```
